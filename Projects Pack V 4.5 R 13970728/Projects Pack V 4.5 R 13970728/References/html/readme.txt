Please put in this folder the html of all web pages you have used as your references.

Rename each file to the first five words of the web page title.

for example: if the title of the web page is

"Optimization of aerodynamic and acoustic performances of supersonic civil transports"

then the file name will be

"Optimization of aerodynamic and acoustic.htm"

There would also exist a folder entitled

"Optimization of aerodynamic and acoustic_files"

including the relating files of the web page.
